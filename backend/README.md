# How to use?

`.env`ファイルを作成し、以下の環境変数を各々の環境の値に設定

```
SOCIAL_AUTH_GOOGLE_OAUTH2_KEY='Google API OAuth2.0 Client ID'
SOCIAL_AUTH_GOOGLE_OAUTH2_SECRET='Google API OAuth2.0 Client Secret'
SOCIAL_AUTH_GOOGLE_OAUTH2_REDIRECT_URL='Google API OAuth2.0 Redirect URL'
SOCIAL_AUTH_GOOGLE_OAUTH2_SCOPE='Google API OAuth2.0 Scope'
SECRET_KEY='Django Secret Key'
JWT_SECRET_KEY='JET Secret Key'
JWT_PRIVATE_KEY='Path to Private Key File'
JWT_PUBLIC_KEY='Path to Public Key File'
```
